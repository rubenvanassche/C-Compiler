# C to P Compiler
This was(is) an project for the course Compilers. The goal was to write a compiler in Python using ANTLR.

## Optional Features
- For loops

## Not implemented
-printf
-scanf


## Building
```
make configure
make generate
```

## Running
```
python3 Compiler.py
```

## Testing
```
make test
make testold
make testcompiler
```
